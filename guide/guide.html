本文会帮助你从头搭建一个简单的 VuePress 文档。如果你想在一个现有项目中使用 VuePress 管理文档，从步骤 3 开始。

  1. 创建并进入一个新目录
    mkdir vuepress-starter && cd vuepress-starter
  2. 使用你喜欢的包管理器进行初始化
    yarn init # npm init
  3. 将 VuePress 安装为本地依赖 我们已经不再推荐全局安装 VuePress
    yarn add -D vuepress # npm install -D vuepress
